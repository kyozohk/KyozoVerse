# KYPRO-36

Actual result: Drag & drop is non-functional in all file upload drop zones across the app. The drop zone shows no hover highlight when a file is dragged over it, no visual response on drop, no filename confirmation, no loading indicator, and no error message. Users who attempted to upload files during QA testing had to use the 'Click to browse' file picker instead in all cases. This affects both the Integrations > Import wizard (CSV and Excel flows) and the Audience > Automatically Integrate modal.
