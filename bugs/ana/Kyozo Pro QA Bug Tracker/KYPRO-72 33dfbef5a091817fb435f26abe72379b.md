Category: UX

Expected result: There should be a single, clear navigation element to go back. Having two controls for the same action creates visual clutter and may confuse users about whether they do different things.
Notes: Confirmed during negative testing on community t1. Consider removing the 'Back' button and keeping only the '← Back to sources' breadcrumb-style link, which is more consistent with the rest of the modal flow.
